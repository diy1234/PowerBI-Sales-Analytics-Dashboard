# Sales Performance Analytics Dashboard

## Project Overview

Developed an interactive Power BI dashboard using the Superstore dataset to analyze sales, profit, customer behavior, and regional performance.

## Tools Used

* Power BI
* DAX
* Power Query
* Excel

## Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Total Quantity
* Profit Margin %
* Average Order Value

## Dashboard Features

* Monthly Sales Trend Analysis
* Regional Performance Analysis
* Category & Sub-Category Analysis
* Top Customers by Sales
* Top Products by Sales
* Interactive Slicers

## Business Insights

* West Region generated highest sales.
* Technology category contributed maximum revenue.
* Phones emerged as the top-selling sub-category.
